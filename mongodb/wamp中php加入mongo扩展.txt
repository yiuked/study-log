PHP连接mongo有两种方式，一种是mongo,一种为mongodb

装的是WampServer Version 2.5 64位版本
Apache 2.4.9
PHP 5.5.12

http://pecl.php.net/package/mongo
在安装php_mongo.dll扩展时，下载的64位版本,
NTS和TS都试过，就是不成功，最后换成32位版本的，竟然通过了.
extension=php_mongo.dll
安装完成，可参照以下链接进行使用：
http://php.net/manual/zh/book.mongo.php


php_mongodb 用于php5.4(5.4才开始支持命名空间)及以上的版本
1.下载php mongodb扩展
http://pecl.php.net/package/mongodb/
复制到ext目录
修改php.ini文件，加入
extension=php_mongodb.dll
确保phpinfo中输出的是php mongodb而不是php mongo

2.加载mongodb
composer require "mongodb/mongodb=^1.0.0"

安装完成，可参照以下链接进行使用:
http://php.net/manual/zh/set.mongodb.php

以下为两种连接mongo并读取数据的方式
<?php
#方式1
$m = new MongoClient("mongodb://zsjr:zsjr@localhost:27017/zsjr");
$db = $m->selectDB('zsjr');
$users = $db->selectCollection('users');
$result = $users->find();
var_dump(iterator_to_array($result));

#方式2
require 'vendor/autoload.php';
$m = new MongoDB\Client("mongodb://zsjr:zsjr@localhost:27017/zsjr");
$users = $m->zsjr->users;
$result = $users->find();
var_dump($result);
