## 如何动态添加一个Org

### 创建
```
PeerOrgs:
  - Name: Org1                 # 组织名
    Domain: org1.example.com   # 组织域名
    EnableNodeOUs: true        # 设置为true时，会在证书目录生成config.yaml
    Template:
      Count: 1                 # 生成的节点数
    Users:
      Count: 1                 # 生成的用户数
```
