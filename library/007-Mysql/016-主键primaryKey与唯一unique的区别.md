|             | 唯一性 | 是否允许为空 | 可以有多少个 | 能否组合 |
| ----------- | ------ | ------------ | ------------ | -------- |
| primary key | √      | ×            | 只有一个     | √        |
| unique      | √      | √            | 可以有多个   | √        |

> * 主键只能有一个，主键可以多个字段建立组合主键（组合主键，只有参与组合的键值都一样时才会冲突，否则任何单一一个字段相等都不会冲突）
> * unique 可以有多个
